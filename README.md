# Deno Expolanets

This project parses data from the nasa keplet exoplanet csv using 
[Deno][deno] and [Lodash][lodash].


## Dependencies

- [Git][]
- [Deno][]

## Build instructions

```sh
# Clone this repo
git clone https://github.com/dmarcr1997/UdacityCPPCapstone.git
cd	denoExoplanets
# Build/Run
deno run --allow-read mod.ts
```
